# Whole Life Insurance Pricing Model with Health & Critical Illness riders

## Project Overview
This project presents a spreadsheet-based pricing model for a whole life insurance product with Health and Critical Illness riders. The model estimates insurance premiums by applying actuarial concepts such as present value calculations, mortality assumptions, lapse rates, and expense loading. It provides a structured framework for analyzing premium calculations and understanding the pricing process for life insurance products.

## Objectives
- Build a practical actuarial pricing model for a whole life insurance product.
- Demonstrate the calculation of net and gross premiums using actuarial techniques.
- Analyze the effects of key pricing assumptions, including mortality, lapse, interest rates, and expenses.
- Develop a transparent spreadsheet model that clearly documents the pricing process.
